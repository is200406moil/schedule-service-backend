# Отчёт (LaTeX)

Сборка PDF выполняется **отдельно** от приложения. Замените преамбулу в `main.tex` на шаблон [latex-g7-32](https://github.com/latex-g7-32/latex-g7-32) и добавляйте файлы в `sections/` и `bib/`.

Рекомендуемые артефакты сборки держать вне репозитория или в игнорируемой папке (см. корневой `.gitignore`).
